# ebetron.github.io

**Personal Resume Website of Ebenezer Rodriguez Vidal**

🌐 Live site: [https://ebetron.github.io](https://ebetron.github.io)

---

## Overview

A static, single-page resume website for Ebenezer Rodriguez Vidal, a Software Engineer and Engineering Leader. The site is hosted via GitHub Pages and presents a clean, mobile-responsive personal resume.

---

## Repository Structure

```
ebetron.github.io/
├── index.html          # Main (and only) HTML page — the full resume
├── CNAME               # Custom domain configuration for GitHub Pages
├── css/
│   └── styles.css      # Core stylesheet (includes Bootstrap customization)
├── js/
│   └── scripts.js      # Core JavaScript (Bootstrap interactions, smooth scroll)
└── assets/
    └── img/
        ├── profile.jpg  # Profile photo displayed in the sidebar
        └── favicon.ico  # Browser tab icon
```

- **Fixed sidebar navigation** with smooth-scroll links (desktop) and a collapsible hamburger menu (mobile)

- **Custom domain** configured via the `CNAME` file

## How to Run Locally

No build tools or dependencies required. Simply clone the repo and open the HTML file in a browser:

```bash
git clone https://github.com/ebetron/ebetron.github.io.git
cd ebetron.github.io
open index.html   # macOS
# or double-click index.html in your file explorer
```
